# Point

- zig 不包含垃圾回收器

- 面向内存编程
- 指针、堆分配、悬挂指针（悬垂指针）

- 数据与内存联系起来
- 变量是将一种类型与特定内存位置联系起来的标签

- 变量指向结构的起点
- 字段是按顺序排列的

- zig不保证结构的内存布局
- packed struct

- 取地址运算符`&`
  - 返回一个变量的地址

- 类型T的值的地址是`*T`

- 结构体方法

- 常量函数
  - 默认情况下，zig会传递一个值的副本（按值传递）

- 小类型可以通过值传递（复制），大类型通过引用传递能方便

- 编译器必须权衡复制的代价和通过指针间接访问字段的代价

- 指向指针的指针

- 嵌套指针

- 递归结构
